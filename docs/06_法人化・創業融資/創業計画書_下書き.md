# 創業計画書 下書き（日本政策金融公庫様式に対応）

公庫の創業計画書は8項目。以下はこの事業の内容で埋めた下書き。
＿＿の箇所と【要記入】は本人しか書けない部分。
数字は販売戦略メモ（セグメント別価格）と整合させてある。

---

## 1. 創業の動機

私立・公立高校の学校徴収金（積立金）会計は、いまも数百人分の手入力と目視照合に
支えられており、担当職員の長時間労働と、保護者からの預かり金を扱う業務としての
ミスのリスクが常態化している。現場でこの実態に接し、既存のExcelマスターを
一切変更せずに入力業務だけを自動化する製品を開発した。実際の学校ファイルでの
検証と導入校での運用を通じて有効性を確認しており、これを全国の学校に提供する
事業として法人化する。

【要記入】現場に接した経緯・原体験を2〜3行で具体的に（面談で必ず聞かれる）

## 3. 取扱商品・サービス

**商品: 学校徴収金（積立金）会計の入力自動化パッケージ「積立金会計 入力アシスタント」**

- 既存のExcelマスターを変更せず、隣に置いた1ファイルで入力を自動化
  （名簿照合・支出/収入一括入力・銀行振替結果の自動照合・承認書自動作成・決算集計・
  精算書PDF・業者別集計・翌年度引き継ぎ、全15機能）
- インストール不要・完全オフライン・生徒情報は校内から出ない設計（公立校でも導入しやすい）
- 販売単価（税別）:
  - 公立高校（単独校）: 年額49,800円（先行導入協力校は年額29,800円・3校限定）
  - 私立高校: 月額19,800円（年払198,000円）
  - 教育委員会一括: 1校あたり年額30,000円（10校以上）
- 売り: ①導入済み校での実測による作業時間削減データ ②実物ファイルでの検証実績
  ③保守・年度更新・担当者交代時の再指導まで含む伴走型

**セールスポイント**: 校務支援システム（生徒1人あたり月約300円が相場）に対し、
本製品は既存ファイルを活かすことで公立校の少額予算でも導入可能な価格を実現。
競合する専用システムとは市場が重ならない。

## 4. 取引先・取引関係等

- 販売先: 都内公立高校（導入協力校＿校・契約済み／見込み＿校）、私立高校（商談中＿校）
  - 回収条件: 請求書払い・年額一括（公立）／月額または年払（私立）
- 仕入先: なし（自社開発ソフトウェア。物理媒体のUSB等は都度購入）
- 外注先: 税理士（顧問・決算）、必要に応じ法務確認
