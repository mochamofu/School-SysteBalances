# 株式会社設立 → 創業融資までのロードマップ（内部用）

この事業（積立金入力アシスタント）を種として法人を作り、
日本政策金融公庫の創業融資を引くまでの実務手順。
金額・制度は2026年7月時点の情報。申請前に公庫・自治体の最新条件を必ず確認する。

---

## 全体像（約2〜3ヶ月）

```
今月        Phase 0  準備（自己資金の整理・有償契約の実績づくり・商号決定）
+2〜4週     Phase 1  株式会社設立（定款→登記→届出→法人口座）
登記直後〜  Phase 2  創業融資申請（公庫 新規開業・スタートアップ支援資金）
+3週〜1.5ヶ月  面談 → 融資決定 → 着金
以後        Phase 3  資金管理と2本目（自治体制度融資）の検討
```

**最重要の順番**: 長山高校の有償成約（29,800円でよい）を先に取る。
「売上のある創業」と「売上ゼロの創業」では審査の景色がまったく違う。

---

## Phase 0: 準備（今月からできること）

1. **自己資金の整理**
   - 融資審査は通帳の履歴を見る。コツコツ貯めた形跡が最強、直前の大金移動（見せ金）は即アウト
   - 目安: 借入希望額の1/3。300万円借りたいなら自己資金100万円
   - 制度上の自己資金要件は撤廃済みだが、実務では依然として審査の中心
2. **実績の証拠を揃える**（このリポジトリの成果物がそのまま使える）
   - 有償契約書・覚書（導入協力校覚書）
   - 製品パッケージv1.0・検証ログ・実測データ（導入前後の作業時間）
   - LP・提案書 ― 「思いつき」ではなく「もう動いている事業」を示す
3. **商号・本店所在地・事業目的を決める**
   - 事業目的の例: ①ソフトウェアの開発・販売・保守 ②業務効率化に関するコンサルティング
     ③教育機関向けサービスの企画・提供 ④前各号に附帯関連する一切の事業
   - 本店は自宅可（賃貸は貸主の承諾を確認）。バーチャルオフィスは口座開設で不利になる場合あり

### Phase 0-b: 資本金・自己資金が足りない場合の選択肢

**前提の整理**: 「資本金」（設立時に入れる額）と「融資審査の自己資金」は別物。
資本金は法律上1円から設立できる。無理にかき集める必要はない。

1. **資本金は手持ちの範囲で**（目安30〜50万円）
   - 10万円未満だけは避ける（法人口座審査・取引先の見た目で不利）
   - 後から増資できるので、最初に積む必要はない
2. **借入希望額を自己資金に合わせて下げる**
   - 1/3ルールの逆算: 自己資金30万なら初回は100万円前後の申請
   - 本事業は設備投資ほぼゼロなので100万でも回る。「初回小さく→返済実績→追加融資」が公庫の王道
3. **融資の時期を後ろにずらして実績で補う**
   - 制度は創業後おおむね7年以内まで使える。急がなくてよい
   - 設立→有償契約→法人口座に毎月入金の通帳を6〜12ヶ月分作ってから申請。
     売上実績は自己資金の弱さを相殺する最強の材料
4. **今日から事業用口座に毎月定額を積む**
   - 審査が見るのは残高ではなく履歴。月3〜5万でも「毎月コツコツ」が評価される
   - 申請直前の一括入金は見せ金と判定されて即アウト
5. **補助的な選択肢**
   - 親からの贈与: 贈与契約書＋出所の説明ができれば自己資金と認められることが多い
     （直前一括は印象が悪い。早めに動かして寝かせる）。親族からの「借入」は負債扱いで自己資金にならない
   - 自治体の制度融資: 自己資金の扱いが公庫より緩い場合あり（認定支援機関経由が速い）
   - 補助金（持続化補助金・都の創業助成）: 自己資金要件はないが後払い。広告費の回収用と割り切る

**推奨ルート**: 手持ち30〜50万で設立 → 毎月積立開始 → 有償成約の入金を通帳に載せる →
6〜12ヶ月後に100〜200万を申請（自己資金が育っていれば300万に引き上げ）。
断られても信用情報に傷は付かない。半年実績を積んで再申請できる。

## Phase 1: 株式会社設立（2〜4週間）

| 手順 | 内容 | 費用目安 |
|---|---|---|
| 1. 定款作成 | freee会社設立／マネーフォワード会社設立で自力作成可 | 0円（サービス利用） |
| 2. 定款認証（公証役場） | **電子定款**にすると印紙代4万円が不要 | 認証手数料 約3〜5万円（資本金による）＋謄本代 |
| 3. 資本金の払込 | 発起人個人口座に振込んで通帳コピー | ― |
| 4. 法務局へ登記申請 | 登録免許税は 資本金×0.7%（最低15万円） | **15万円** |
| 5. 登記完了（約1〜2週間） | 履歴事項全部証明書・印鑑カード取得 | 数千円 |
| **設立費用 合計** | 電子定款・自力なら | **約19〜21万円** |

- **資本金の目安: 理想は100万〜300万円。足りなければ30〜50万円でも可（Phase 0-b参照）**
  - 1,000万円未満 → 設立当初の消費税免税・住民税均等割が最低額（年7万円）
  - 10万円未満だけは避ける（融資・口座開設で印象が悪い）。手持ちの自己資金をそのまま資本金にするのが素直
- 司法書士に依頼する場合は＋5〜10万円（時間を買う選択肢）

**設立直後にやる届出（期限あり）**
- 税務署: 法人設立届出書（2ヶ月以内）／**青色申告承認申請（設立3ヶ月以内・必須）**／
  給与支払事務所開設届／源泉所得税の納期特例申請
- 都道府県税事務所・市区町村: 法人設立届
- 年金事務所: 社会保険新規適用（役員報酬を出すなら必須）
- 法人口座 → 下記「法人口座の作り方」参照

### 法人口座の作り方（推奨: 信金メイン＋ネット銀行サブ）

**メイン: 川崎信用金庫**（個人事業の口座がある取引先）
- 既存の取引実績があるため、新設法人でも口座審査に通りやすい（ゼロから申し込むメガバンクとの最大の違い）
- 信用金庫は**制度融資（信用保証協会付き・利子補給）の窓口**。
  公庫を1本目、信金経由の制度融資を2本目とする二段構えの布石になる
- 公庫の借入申込書「取引金融機関」欄に記載でき、着金口座にも指定できる
- 注意: 信金には営業地区の会員資格制約がある。法人の本店所在地が営業エリア内であること
- 段取り: 登記完了→謄本・法人印鑑証明を持って**いつもの窓口へ**
  「法人成りした。法人口座と創業融資の相談を」と持ち込む。個人事業の履歴が最強の紹介状

**サブ: ネット銀行**（GMOあおぞら・住信SBI等）
- 開設が速く（最短数日）、振込手数料が安い。日々の支払い・経費振込用
- オンライン申込ではLP・製品資料・契約書の提示が効く（この事業は全部ある）

**メガバンクの法人口座は作らない**
- 新設法人に厳しく（審査数週間・謝絶あり）、取りに行く理由がない
- 取引先が公立学校なら地元信金の口座で問題ない（学校事務にも馴染む）
- 個人のメガバンク口座は普段使いのまま。役員報酬の振込先にする

**お金の流れの固定（公私分離が融資と税務の基本）**
- 学校からの入金・融資着金・税金社保 ＝ 信金法人口座
- 日々の支払い・振込 ＝ ネット銀行
- 生活費 ＝ 役員報酬として月1回定額を法人口座から個人口座へ
- 個人事業時代の事業口座は、入金先の切り替え完了後に整理（既存取引先には振込先変更を案内）

## Phase 2: 創業融資（日本政策金融公庫）

**使う制度: 新規開業・スタートアップ支援資金**（旧・新創業融資制度は2024年3月廃止）
- 対象: 新たに事業を始める方、または事業開始後おおむね7年以内
- 融資限度額: 7,200万円（うち運転資金4,800万円）― 実務の創業初回は数百万円台が中心
- **原則、無担保・無保証人**（法人代表者の連帯保証も不要にできる）
- 自己資金要件: 制度上は撤廃。実務は自己資金の1/3ルールで考える
- 利率: 数％台（条件・情勢で変動。女性・35歳未満・55歳以上等の優遇区分あり）

**申請額の設計（この事業の場合）**
- 希望額の目安: **300万円**（自己資金100万円前提）
- 資金使途（運転資金中心・設備はほぼ不要な事業）:
  - 営業活動費（交通費・USB等資材・展示会/研究会参加）: 60万
  - 広告宣伝費（LP改修・印刷物・Web広告テスト）: 60万
  - 外部委託費（法務確認・税理士顧問・デザイン）: 60万
  - 当面の役員報酬・予備費（売上が立つまでの12ヶ月分の一部）: 120万
- 設備資金がほぼゼロの計画は「何に使うのか」を細かく聞かれる。上記の内訳を数字で言えるようにする

**流れ**: 事業資金相談ダイヤル/支店窓口 → 申込（インターネット申込可）→
必要書類提出 → **面談（1〜2週間後・30分〜1時間）** → 現地確認 → 決定 → 契約 → 着金。
全体で3週間〜1.5ヶ月。

**必要書類**
- 創業計画書（公庫様式）― 下書き: `創業計画書_下書き.md`
- 見積書（資金使途の根拠）、通帳コピー（自己資金）、
  本人確認書類、法人の履歴事項全部証明書、（賃貸なら）賃貸借契約書
- **任意で出すと効く追加資料**: 有償契約書・LP・製品パッケージ・検証ログ・実測データ

**面談の勝ち筋（この事業の強み）**
1. 有償契約が既にある（審査官が一番見たいのは「売れるのか」への答え）
2. 実物ファイルで検証済みの製品が完成している（開発リスクが残っていない）
3. 限界費用が極小（USB＋送料1,500円/校）で、売上≒粗利の構造
4. 経営者の実務理解（現場の業務フローを説明できること）― 面談で必ず自分の言葉で話す

## Phase 2b: 並行して検討する制度

- **自治体の制度融資**（東京都・区市の創業融資）: 信用保証協会付き・利子補給や保証料補助あり。
  公庫と併用可。認定経営革新等支援機関（税理士等）経由だと段取りが速い
- 小規模事業者持続化補助金（創業枠）: 融資ではなく補助金。LP・販促費の一部に充当可能
- 東京都・区の創業助成金: 要件が合えば併願

## Phase 3: 融資後

- 資金使途どおりに使う（転用は次の融資が死ぬ）。法人口座で入出金を完結させ、月次試算表を作る
- 1期目の実績（学校数・売上）を作り、2年目に自治体制度融資や公庫の追加融資で増枠
- 税理士顧問（月1〜3万円）は融資後すぐ付けるのが結局安い（決算・年末調整・融資資料）

## 注意（つまずきポイント）

- 見せ金・タンス預金の一括入金は自己資金と認められない
- 法人設立**前**でも公庫は申込可能だが、法人でやると決めているなら登記後に法人名義で申し込むのが素直
- 代表個人の信用情報（カード延滞・携帯割賦の遅延）は事前にCICで自己開示して確認
- 「起業アイデア段階」に見られたら負け。**製品・契約・データが揃った状態で申し込む**（＝長山成約後）
