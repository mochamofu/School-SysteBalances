# docs/ の案内（シーン別）

資料は「どの場面で使うか」で番号フォルダに分かれています。

## 01_学校向けマニュアル/ … 導入済みの学校に渡す・置いてくる資料

| ファイル | 用途 |
|---|---|
| 使い方ガイド_図解版.pdf | 事務担当者向けの図解マニュアル。印刷して保管用PCの横に置く想定 |
| 入力アシスタント_手順書.pdf | 機能別の詳細手順書（①〜⑮の操作） |

## 02_営業・商談資料/ … 商談・稟議で使う資料

| ファイル | 用途 |
|---|---|
| 業務改善提案書.pdf | お客様配布用の提案書 |
| 価格とサービスのご説明.pdf | 価格比較のA4一枚。稟議添付用（その場で読み上げず置いてくる） |

| 料金設計メモ.md | 【内部用】月額9,800/19,800/34,800円の算定根拠・市場相場 |

## 03_現地デモ・レクチャー/ … 訪問してデモするとき

| ファイル | 用途 |
|---|---|
| デモ実施手順書.pdf | 2台PCデモの進め方（USBで持ち歩くデモキットとセット） |
| 動作確認_2台PCシミュレーション手順書.pdf | 実機での動作確認手順 |

デモキット本体は `python assistant/src/make_usb_kit.py` で生成
→ `assistant/output/積立金入力アシスタント_デモキット.zip`

## 04_オンライン導入/ … 郵送USB＋Zoomで導入するとき（全国展開）

## _build/ … 上記PDF・スライドの生成スクリプト

`build_price_onepager.py`（→02）、`build_online_onboarding.py`（→04）、
`build_slides.py`（→02）などを実行すると、対応するフォルダに再生成されます。
docx→PDF変換にLibreOffice（soffice）を使用。

## animations/ … 操作アニメーション（GIF）

`index.html` をブラウザで開くとまとめて見られます。
`2台PCシミュレーション.gif` が現行製品のデモ動画。①〜④のGIFは第1世代ツールのもの。
生成スクリプト（`_engine.py`・`anim*.py`）も同じフォルダにあります。
